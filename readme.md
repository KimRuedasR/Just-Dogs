# ­ЪљЋ Just Dogs App ­ЪљЋ

## ­ЪЊќ­ЪљЙ Description 

Just Dogs is a mobile application dedicated to dog lovers. Inspired by the concept of Instagram, this app is a niche social network where users can share and view photos of dogs. The unique feature of this app is its use of machine learning to recognize and feature only dog pictures, ensuring a focused and delightful experience for canine enthusiasts.

## ­ЪЏа№ИЈ­ЪљЋ Requirements 

- Node.js
- React Native
- Firebase
- Machine Learning API (for dog recognition)

## ­Ъџђ­ЪљЙ Setup 

### 1. Clone Repo ­ЪЊЦ

```bash
git clone https://github.com/KimRuedasR/Just-Dogs.git
```

### 2. Install Dependencies ­ЪЊд

Navigate to the project directory and run:

```bash
npm install
```

## 3. Firebase Setup ­ЪћЦ

Set up Firebase in the project for authentication and database services. Follow the instructions in the Firebase documentation to integrate it into the React Native app.

### 4. Start the App with Expo ­ЪїЪ

```bash
npm start
```
